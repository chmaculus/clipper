!short: GKI_PMouseDisable          Disable the mouse
'------------------------------------------------------------------------------
void GKI_PMouseDisable()

Disables the mouse. Call this function before exiting a 
program to remove interrupt routines.

!seealso: "GKI_PMouseEnable" 
'------------------------------------------------------------------------------



!short: GKI_PMouseEnable           Enable the mouse
'------------------------------------------------------------------------------
int GKI_PMouseEnable() => Number of buttons

Enables the mouse, resets it, and returns the number of 
buttons. Installs interrupt service routines that must be 
removed by calling the function GKI_PMouseDisable() before 
quitting an application.

^BReturns^B
Zero if the mouse cannot be found. Otherwise, an integer 
indicating the number of buttons on the mouse.


!seealso: "GKI_PMouseDisable" 
'------------------------------------------------------------------------------
