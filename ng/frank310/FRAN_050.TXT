     sites.

 2.  Demos of ^bADhotbar()^b, a new engine added in v3.00, were added.

     itself displays a number of text files.  In older versions, this was
     accomplished using ADfview().  V3.00 uses ADfshow().

     ^ban array^b and subsequently manipulate the array to view and print the
     report was added.  Contributed by David L. Smith.
